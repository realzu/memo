# DOM

> 문서 객체 모델 (Document Object Model)

HTML, XML 문서의 콘텐츠 및 구조 등을 계층화해서  
문서를 읽고 조작할 수 있도록 하는 API(프로퍼티, 메서드)를 제공하는 트리 자료구조  
문서 내의 모든 요소를 정의하고, 각각의 요소에 접근하는 방법을 제공  
(자바스크립트가 이를 이용해 HTML 요소 추가, 스타일 수정 등이 가능)
